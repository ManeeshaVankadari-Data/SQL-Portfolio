# SQL Portfolio: Core to Advanced Queries

Welcome to my SQL project portfolio. This repository serves as a practical showcase of my database management, structural logic, and analytical querying skills using MySQL.

---

## 📁 Repository Showcase

* **`SQL-Syntax-Reference/`**
    * `parks_and_rec_queries.sql`: A master script built on a relational database model to demonstrate comprehensive SQL capabilities from fundamental data retrieval to advanced programmatic automation.
* **`Project-2-Data-Cleaning/`** 
    * `layoffs_data_cleaning.sql`: A complete data cleaning and standardization project using a global layoffs dataset. Handles row deduplication via CTEs, text trimming, date formatting conversion, NULL value backfilling via self-joins, and data reduction.
    * `layoffs.csv`: The raw dataset used for the project.
* **`Project-3-Data-Exploration/`**
    * `layoffs_exploratory_analysis.sql`: An advanced exploratory data analysis project conducting extensive multi-level queries. Highlights time-series trends using rolling metrics, nested yearly performance ranking filters, corporate funding analysis via group aggregations, and industry-specific macroeconomic trend shifts using window functions.

---

## 🛠️ Key Technical Highlights

This portfolio demonstrates a strong technical command over database development, including:

* **Advanced Joins & Sets:** `INNER`, `LEFT`, `RIGHT`, `CROSS`, and `SELF` joins, along with dataset combinations using `UNION` / `UNION ALL`.
* **Analytical Windows:** Complex ranking and running calculations via `PARTITION BY`, `ROW_NUMBER()`, `RANK()`, and `DENSE_RANK()`.
* **Performance Optimization:** Streamlining complex queries using **Common Table Expressions (CTEs)**, **Views**, and **Temporary Tables**.
* **Database Automation:** Building backend logic via **Stored Procedures** with parameters, automated **Triggers**, scheduled **Events**, and custom **Functions**.
* **Control Flow & Loops:** Processing data row-by-row using database **Cursors** and `CONTINUE HANDLER` conditions.
